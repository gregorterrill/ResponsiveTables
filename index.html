Responsive Tables

by @GregorTerrill

This jQuery script applies column headers to each sub-item at a pre-defined breakpoint, allowing more data to be displayed on mobile devices with a limited screen width.

This demo uses debounced window resizing by @louis_remi to decrease the frequency of resize events across different browsers.

Each table that is to be made responsive requires the class name responsiveTable. Tables need to be properly scoped in order for the color swapping to function correctly.

The default breakpoint is 500px.

Resize the browser window to see the change occur.
